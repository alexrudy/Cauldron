Abstract Base Classes for KTL
=============================

The KTL API is defined in :mod:`Cauldron` by four abstract base classes, :class:`~client.Service`, :class:`~clinet.Keyword`, :class:`~dispatcher.Service` and :class:`~dispatcher.Keyword`

These classes are designed to provide a consistent API for KTL services and keywords, irrespective of the KTL backend.
