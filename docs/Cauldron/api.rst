Abstract Base Classes for KTL
=============================

The KTL API is defined in :mod:`Cauldron` by two abstract base classes, :class:`~service.Service` and :class:`~keyword.Keyword`.

These classes are designed to provide a consistent API for KTL services and keywords, irrespective of the KTL backend.
